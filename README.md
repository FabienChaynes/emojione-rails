# Emojione::Rails

Sprocket/Rails emojione package, see http://emojione.com

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'emojione-rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install emojione-rails

## Usage

In your JavaScript,Css manifest (e.g. `application.js`):

    //= require emojione
